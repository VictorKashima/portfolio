Descrição do projeto

O TP Link GT é um carrinho de controle remoto que funciona em 3 modos, o primeiro modo é o Stand By para poupar energia, o segundo é o modo de pilotagem Automática que funciona seguindo uma linha de fita isolante e o terceiro é o modo Manual controlado por raios infra vermelho. Este projeto foi o TCC do meu grupo no curso técnico de eletroeletrônica.

Minha participação

  • Programação do arduino (C++).
  • Dimensionamento e escolha dos componentes.
